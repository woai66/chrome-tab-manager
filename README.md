# Tab Manager — Chrome Extension

侧边栏标签页管理插件，支持分组和休眠功能。

## 功能

- 侧边栏展示当前所有标签页（垂直列表）
- 创建/重命名/删除标签组
- 将标签"休眠"到分组（关闭标签但保留 URL）
- 一键恢复休眠标签
- 数据持久化，重启浏览器后分组不丢失

## 安装

1. 克隆本仓库
2. 打开 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目目录

## 要求

Chrome 114+（需要 sidePanel API 支持）
